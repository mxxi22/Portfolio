Gelateria MAKO

Webサイト制作

概要 Overview

新宿駅のファッションビルに新たにオープンするジェラート店「Gelateria MAKO」（模擬課題）

  • 制作期間：10日間（2020年3月）
  • 担当箇所：デザイン/コーディング（レスポンシブ）
  • 使用ツール：illustrator / Photoshop / Figma / HTML / CSS / jQuary
  • URL：https://

制作プロセス Proccess

クライアント調査・情報整理

競合分析

ポジショニングマップ（他社様のロゴを使用しているため、ロゴ抜きで掲載）

都内のジェラート販売店を調査し、ポジショニングマップ上に置いておくと、以下のような傾向が見えてきます。

・高価格のお店ほど、デザインが洗練されている印象がある
・ジェラート業界では、あまりwebサイトのデザインに力を入れておらず、そこを狙うことでおしゃれ好きやこだわり志向の女性の流入を目指せる

アイデア発散

マインドマップ（ウェビング）

マインドマップを作成して100単語以上書き出し、そこからお店の特徴ややターゲットの好みに合いそうなキーワードを抽出しました。

キーワード抽出

コンセプトの決定

コンセプトの解説

Gelateria MAKOはこだわり志向の高級ジェラート店であることがわかり、またターゲットの好みとキーワードから、シンプルで洗練された、落ち着いたデザインが合っていると予想しました。

制作物 Works

◾こだわりポイント1

ファーストビューは、落ち着いた色味で余白が多いジェラートの写真を選定しています。 写真、ロゴ、キャッチコピーのみのシンプルなデザインで、落ち着いた雰囲気を意識しました。 また、高級感が出るよう、余白や行間の調整にもこだわりました。

◾こだわりポイント2

メニューはユーザーが一番気になる部分と考え、分かりやすさを重視してシンプルなデザインにしています。 見やすさを重視し、余白にこだわりました。

◾こだわりポイント3

クライアント様の特徴であるジェラートのなめらかさが伝わる写真を背景として使用し、スルーしがちなニュースも見てもらえるように工夫しています。
